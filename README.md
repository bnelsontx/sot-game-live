# Source of Truth or Consequences

[Play LIVE Here!](https://bnelsontx.github.io/sot-game-live/)

An idle game by [bnelsontx](https://github.com/bnelsontx). Inspired by the real pain (and joy) of network automation.

> You have a laptop, a console cable, and 47 devices that were last documented in a spreadsheet that Kevin started in 2019. Kevin doesn't work here anymore.
>
> Nobody knows the enable password on the core switch.

## How to Play

Open `index.html` in a browser. That's it. No server required. No dependencies. No 47-character API token. No Change Advisory Board approval.

You can play the live game at: [https://bnelsontx.github.io/sot-game-live/](https://bnelsontx.github.io/sot-game-live/)

## What Is This

You start as a lone network engineer typing `show run` into a dusty console port. You end up commanding a fully automated, self-healing multi-site network empire. Every step in between is painfully, hilariously real.

## Learn While You Play

Every building, technology, worker role, and upgrade in the game corresponds to a real network engineering concept. Look for the **ℹ** button on items throughout the game — it opens a short, serious description of the real-world technology behind the game element. You'll learn about protocols (SSH, BGP, OSPF), tools (Ansible, Nornir, Nautobot), practices (GitOps, ZTP, CI/CD), and roles (SRE, NOC Technician, Automation Architect) as you progress through the six eras of network engineering maturity.

## Quick Start Guide

1. **Pour yourself a coffee.** You're going to need it. Your engineers will too.
2. **Read the [User Guide](https://bnelsontx.github.io/sot-game-live/user-guide.html).** Yes, really. We know you never read documentation — that's half the reason your network looks like this. The guide is short, funny, and contains fewer lies than your current topology diagram.
3. **Click the button.** You know the one. It says `> show version`. Click it a few more times. Feels good, right? That's the most productive you'll be all day.
4. **Accumulate CLI Commands.** Just like real life, except here they're useful.
5. **Build things.** Terminal emulators, coffee machines, syslog servers nobody reads.
6. **Hire engineers.** They consume coffee at an alarming rate. Also like real life.
7. **Research technologies.** Progress from "I can automate this with a bash script" to "the network has achieved consciousness."
8. **Try not to lose your Sanity.** (The resource. And also yours.)

## Requirements

A web browser. Literally any web browser made after 2020. If you're running IE11, you have bigger problems than this game can address — but honestly, that tracks for this industry.
